# wc-rs
Counting words with Rust. This project is simply for learning purposes.

## Building
```sh
$ cargo build --release
```

## Usage

```sh
$ target/release/wc-rs ~/path/to/words_dict.txt ~/path/to/input.txt <number_of_threads>

the 27774997
to 20896481
a 17984751
of 14939545
and 13697946
is 11471216
that 10235401
in 9217977
for 7262176
you 7061003
```
